asdcplib-rs

Safe Rust readers and writers for JPEG 2000, PCM, timed text, and Atmos MXF files. The low-level crate builds CineCert asdcplib through CMake.

Verification

The integration suite writes a PCM MXF, reads its descriptor and frame back, and checks the bytes through the real C++ boundary.

git clone --recursive https://github.com/PostPerfection/asdcplib-rs.git
cd asdcplib-rs
cargo test
cargo clippy --all-targets -- -D warnings
